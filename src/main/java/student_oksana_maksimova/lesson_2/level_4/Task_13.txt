Найти в интеренете ответ на вопрос:
Как откомпилировать код с помощью JDK компилятора javac,
без помощи среды разработки (из консоли)?

Написать исходный код в любом текстовом редакторе, сохранить.
Файл с исходным кодом должен иметь расширение .java и иметь в точности такое же имя, как и имя публичного (public) класса.
В командной строке переходим в папку, где лежит файл (или в самой папке правой кнопкой мышки > открыть в терминале).
 Компилятор Java преобразовывает файл с исходным кодом Java в файл с байткодом Java.
 Следующая команда компилирует НашФайл.java:

javac НашФайл.java

Если нет синтаксических ошибок, компилятор генерирует файл байкода с расширением .class.
Следовательно, приведённая выше команда генерирует файл с названием НашФайл.class.
