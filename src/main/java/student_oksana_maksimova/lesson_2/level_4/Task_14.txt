Найти в интеренете ответ на вопрос:
Как запустить откомпилировнный код с помощью JDK и виртуальной
машины Java (JVM), без помощи среды разработки (из консоли)?

 Следующая команда выполняет байткод для программы, которая приведена выше:

java НашФайл

Внимание: не используйте расширение .class в команде, когда запускаете программу. Используйте ИмяКласса для запуска программы.
Если вы в командной строке используете ИмяКласса.class, то система будет пытаться работать с файлом ИмяКласса.class.class.
