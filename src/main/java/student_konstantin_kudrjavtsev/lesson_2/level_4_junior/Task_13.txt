Найти в интеренете ответ на вопрос:
Как откомпилировать код с помощью JDK компилятора javac,
без помощи среды разработки (из консоли)?

Идем в папку, куда IDEA сохранила проект. Находим там наш файл. Консоль вызывается так: щелкаем правой клавишей
мыши с зажатой клавишей Shift на пустом месте в папке, где лежит файл нашей программы, и выбираем пункт
контекстного меню «Открыть окно команд».
Для того, чтобы скомпилировать его нужно набрать в консоли команду javac и в качестве параметра передать имя нашего
файла.
javac (или  javac.exe) - В первом случае будет вызван компилятор
java (или  java.exe) - стартует наша программа
