Найдите в интеренете ответ на вопрос:
Как узнать какая версия JDK установлена на вашем компьютере?


1. Нажать на значок "яблоко". Он находится в левом верхнем углу экрана (на панели меню). Раскроется меню
2. Выбрать "системные настройки". Эта вторая опция в меню
3. Щелкнуть по значку Java. В новом окне откроется панель управления Java
4. Перейти на вкладку "Обновление". На ней будет  написано версия. У меня версия Java 8 update 361
